# Business Data Analysis Dashboards (Excel)

## Overview
Interactive dashboards built in Excel to analyze business performance and visualize KPIs.  
This project demonstrates skills in PivotTables, PivotCharts, slicers, and advanced formatting.

## Features
- KPI cards for sales, footfall, and averages
- Trend analysis by time and category
- Product performance and store comparisons
- Interactive filters for day and month

## Tools Used
- Excel (PivotTables, PivotCharts, slicers, conditional formatting)

## How to Use
Download the Excel file from this repository and explore the dashboards.  
Use slicers and filters to interact with the data.
